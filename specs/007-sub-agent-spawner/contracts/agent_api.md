# Sub-Agent API Contract

## Module: `simpleclaw.agents`

### SubAgentSpawner

Main interface for spawning and managing sub-agents.

```python
class SubAgentSpawner:
    def __init__(self, config: dict) -> None: ...
    async def spawn(self, command: list[str], task: str, scope: PermissionScope | None = None, timeout: int | None = None) -> SubAgentResult: ...
    async def spawn_python(self, script_path: str, task: str, scope: PermissionScope | None = None, timeout: int | None = None) -> SubAgentResult: ...
    def get_running(self) -> list[SubAgent]: ...
    def get_pool_status(self) -> dict: ...  # {"running": N, "queued": M, "max": 3}
    async def shutdown(self) -> None: ...
```

### ConcurrencyPool

Manages the pool of concurrent sub-agent slots.

```python
class ConcurrencyPool:
    def __init__(self, max_concurrent: int = 3) -> None: ...
    async def acquire(self) -> None: ...
    def release(self) -> None: ...
    @property
    def running_count(self) -> int: ...
    @property
    def available_slots(self) -> int: ...
```

### WorkspaceManager

Creates and manages sandboxed workspace directories.

```python
class WorkspaceManager:
    def __init__(self, base_dir: str | Path, cleanup: bool = False) -> None: ...
    def create(self, agent_id: str) -> Path: ...
    def cleanup(self, agent_id: str) -> None: ...
    def cleanup_all(self) -> None: ...
```
