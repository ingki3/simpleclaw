# Daemon API Contract

## Module: `simpleclaw.daemon`

### AgentDaemon

The main orchestrator class for the daemon process.

```python
class AgentDaemon:
    def __init__(self, config_path: str | Path) -> None: ...
    async def start(self) -> None: ...
    async def stop(self) -> None: ...
    def is_running(self) -> bool: ...
```

### CronScheduler

CRUD operations for cron jobs.

```python
class CronScheduler:
    def add_job(self, name: str, cron_expression: str, action_type: ActionType, action_reference: str) -> CronJob: ...
    def list_jobs(self) -> list[CronJob]: ...
    def get_job(self, name: str) -> CronJob | None: ...
    def update_job(self, name: str, **kwargs) -> CronJob: ...
    def remove_job(self, name: str) -> bool: ...
    def enable_job(self, name: str) -> CronJob: ...
    def disable_job(self, name: str) -> CronJob: ...
```

### HeartbeatMonitor

Tick-based status monitoring.

```python
class HeartbeatMonitor:
    async def tick(self) -> HeartbeatTick: ...
    def get_last_tick(self) -> HeartbeatTick | None: ...
```

### DreamingTrigger

Condition evaluator for automatic dreaming.

```python
class DreamingTrigger:
    async def should_run(self) -> bool: ...
    async def execute(self) -> None: ...
```

### WaitStateManager

Async wait state management.

```python
class WaitStateManager:
    def register_wait(self, task_id: str, state: dict, condition_type: str, timeout: int) -> WaitState: ...
    def resolve_wait(self, task_id: str, resolution: str = "completed") -> WaitState | None: ...
    def get_pending(self) -> list[WaitState]: ...
    def check_timeouts(self) -> list[WaitState]: ...
```

### DaemonStore

SQLite persistence layer.

```python
class DaemonStore:
    def __init__(self, db_path: str | Path) -> None: ...
    # CronJob CRUD
    def save_job(self, job: CronJob) -> None: ...
    def get_job(self, name: str) -> CronJob | None: ...
    def list_jobs(self) -> list[CronJob]: ...
    def delete_job(self, name: str) -> bool: ...
    # Execution logging
    def log_execution(self, execution: CronJobExecution) -> None: ...
    def get_executions(self, job_name: str, limit: int = 10) -> list[CronJobExecution]: ...
    # Wait states
    def save_wait_state(self, state: WaitState) -> None: ...
    def get_wait_state(self, task_id: str) -> WaitState | None: ...
    def get_pending_waits(self) -> list[WaitState]: ...
    def resolve_wait_state(self, task_id: str, resolution: str) -> None: ...
    # Daemon state
    def get_state(self, key: str) -> str | None: ...
    def set_state(self, key: str, value: str) -> None: ...
```
